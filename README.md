# Turkish Straits AIS Intelligence

**Open-Source Intelligence (OSINT) Analysis of Vessel Movements Through the Turkish Straits**

---

## 📌 About This Project

This project is an open-source intelligence analysis of maritime traffic through the Bosphorus and Dardanelles straits using publicly available Automatic Identification System (AIS) data. The analysis focuses on identifying patterns, anomalies, and dual-use cargo movements of strategic interest.

## 🎯 Objectives

- Monitor and analyze vessel traffic patterns through the Turkish Straits
- Identify vessels of strategic or intelligence interest using open-source AIS data
- Assess the geopolitical significance of maritime movements in the region

## 🗂️ Repository Structure

- `/data` — AIS data sources and vessel tracking references
- `/analysis` — Intelligence assessments and written reports
- `/visuals` — Maps, charts, and vessel movement infographics

## 🔍 Methodology

All research is based exclusively on open-source information including AIS tracking platforms, vessel registries, port authority data, and open-source news reporting.

## ⚠️ Disclaimer

This project is purely academic and based entirely on publicly available information.
