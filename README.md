# Turkish Straits AIS Intelligence

### Open-Source Intelligence (OSINT) Analysis of Vessel Movements Through the Turkish Straits

---

## About This Project

This project examines maritime traffic through the Bosphorus and Dardanelles using publicly available Automatic Identification System (AIS) data and existing open-source reporting. The focus is on understanding documented patterns — strait closures, sanctioned-vessel movements, and grain-corridor monitoring — as covered by AIS-tracking platforms, maritime researchers, and investigative outlets, rather than on independently operating a real-time tracking system.

This is an academic/portfolio project. It does not involve original AIS data collection, paid tracking-platform subscriptions, or claims of real-time monitoring capability. Findings are synthesized from publicly published reporting and tracking-platform outputs that are already in the public domain.

## Objectives

- Summarize documented vessel-traffic patterns through the Turkish Straits, as reported by AIS platforms and maritime researchers
- Describe how AIS data has been used by named organizations to identify vessels of strategic interest (e.g., sanctioned tankers)
- Assess the geopolitical significance of maritime movements in the region, grounded in cited reporting rather than independent analysis of raw AIS feeds

## Repository Structure

- `/data` — Links to AIS platforms and vessel-tracking references used as secondary sources
- `/analysis` — Written assessments synthesizing publicly reported findings
- `/visuals` — Planned maps, charts, and vessel-movement infographics (status: not yet produced — see `/visuals/Visuals.md`)

## Methodology

This project relies exclusively on open-source information: free-tier AIS tracking platforms, vessel registries, port authority statements, and open-source/investigative news reporting. No proprietary AIS feeds, satellite tasking, or non-public maritime intelligence were used. Where a finding originates from a single source (e.g., a volunteer-run monitoring account), this is noted explicitly rather than presented as independently confirmed.

## Disclaimer

This project is academic in nature and based entirely on publicly available information. It is not an operational maritime intelligence product and should not be relied upon for real-time vessel tracking, sanctions compliance, or navigational decisions.
